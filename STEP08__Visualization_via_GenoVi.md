#### Patching of installation

##### Patching of `createConf.py`: providing full Circos `etc/` tree
Fix all missing `etc/*.conf` errors permanently by copying `/etc/circos` into the run directory every time GenoVi generates `circos.conf`.

```bash
CONF="$HOME/.local/lib/python3.13/site-packages/scripts/createConf.py"
cp -a "$CONF" "$CONF.bak"

python3 - <<'PY'
import pathlib, re
p = pathlib.Path.home()/".local/lib/python3.13/site-packages/scripts/createConf.py"
txt = p.read_text()

anchor = 'file = open("circos.conf", "w")'
assert anchor in txt

txt = re.sub(
    r'\n\s*# Ensure Circos can resolve includes like etc/.*?\n',
    '\n',
    txt,
    flags=re.S
)

txt = txt.replace(anchor, anchor + '''
    import shutil, os
    shutil.rmtree("etc", ignore_errors=True)
    try:
        shutil.copytree("/etc/circos", "etc")
    except Exception:
        os.makedirs("etc", exist_ok=True)
''', 1)

p.write_text(txt)
PY
```

##### Patching of `createConf.py`: removing all SVG/PNG/file overrides
Prevent Circos “multivalue” errors caused by duplicate `svg`, `png`, and `file` directives.

```bash
python3 - <<'PY'
import pathlib, re
p = pathlib.Path.home()/".local/lib/python3.13/site-packages/scripts/createConf.py"
txt = p.read_text()

txt = re.sub(
    r'\n\s*# Force GenoVi/Circos to generate an SVG.*?pass\n',
    '\n',
    txt,
    flags=re.S
)

p.write_text(txt)
PY
```

##### Patching of `GenoVi.py`: fixing broken Circos shell redirection
Remove invalid shell syntax that caused `Bad fd number`.

```bash
GENOVI="$HOME/.local/lib/python3.13/site-packages/scripts/GenoVi.py"
cp -a "$GENOVI" "$GENOVI.bak"

python3 - <<'PY'
import pathlib, re
p = pathlib.Path.home()/".local/lib/python3.13/site-packages/scripts/GenoVi.py"
txt = p.read_text()

txt = re.sub(
    r'os\.system\("circos -conf circos\.conf.*?"\)',
    'os.system("circos -conf circos.conf -outputdir . -outputfile circos.svg > circos.log 2>&1")',
    txt
)

txt = txt.replace(
    'os.system("circos -debug_group _all >/dev/null 2>&1")',
    'os.system("circos -debug_group _all > circos.debug.log 2>&1")'
)

p.write_text(txt)
PY
```

##### Patching matplotlib style incompatibility: adding a try/except preamble in `create_tables.py`
Replace the single plt.style.use("seaborn") call with a safe fallback.

```bash
TBL="$HOME/.local/lib/python3.13/site-packages/scripts/create_tables.py"
cp -a "$TBL" "$TBL.bak"

python3 - <<'PY'
import pathlib, re
p = pathlib.Path.home()/".local/lib/python3.13/site-packages/scripts/create_tables.py"
txt = p.read_text()

txt2 = re.sub(
    r'plt\.style\.use\("seaborn"\)',
    'try:\n    plt.style.use("seaborn-v0_8")\nexcept OSError:\n    plt.style.use("default")',
    txt,
    count=1
)

p.write_text(txt2)
PY
```

#### Correction of GenBank flatfile generated by Geneious
When exporting the annotated assembly from Geneious in GenBank flatfile format: 
- Select: Convert annotations to strict GenBank format
- Check yes: Generate new _ranslation_ qualifiers for truncated CDS annotations
- Uncheck: Allow names longer than 16 characters

The resulting GenBank flatfile is incomplete. Its header area must be reconstituted and corrected. The following code does this:

```bash
awk '
/^LOCUS/ {l=$2; n=$3}
 /^ACCESSION/ {print "ACCESSION   " l; next}
 /^VERSION/ {print "VERSION     " l ".1"; next}
 /^FEATURES[ \t]+Location\/Qualifiers/ {
     print
     print "     source          1.." n
     print "                     /organism=\"" l "\""
     print "                     /mol_type=\"genomic DNA\""
     next
 }
 {print}
' ./FinalAssembly_Bactopia__input_to_GenoVi.gb > ./FinalAssembly_Bactopia__input_to_GenoVi_corrected.gb 
```
#### Adding translations to GenBank flatfile
For COGs to be identified and visualized by GenoVi, the input GenBank flatfile must contain translation qualifiers for every CDS. Oftentimes, these are not present in the GenBank flatfile. The custom script `add_translations_to_gb.py` adds them. Hence, the script must be executed before genoVi is run.

```
python3 add_translations_to_gb.py \
    -i ./FinalAssembly_Bactopia__input_to_GenoVi_corrected.gb \
    -o ./FinalAssembly_Bactopia__input_to_GenoVi_corrected_withTranslations.gb \
    --table 11 --overwrite
```

#### Execution of GenoVi
```bash
# Removing temporary files from previous run
rm -rf conf etc circos.conf circos.svg circos.png genovi-temp circos.log circos.debug.log 2>/dev/null

genovi -i ./FinalAssembly_Bactopia__input_to_GenoVi_corrected_withTranslations.gb -cs strong -s complete -t FinalAssembly_Bactopia -te --size -k -v verbose

cp ./genovi/genovi.svg FinalAssembly_Bactopia__output_from_GenoVi.svg
```
