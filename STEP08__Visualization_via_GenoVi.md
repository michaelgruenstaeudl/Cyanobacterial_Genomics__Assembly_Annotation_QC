#### Patching of installation

##### Patching of createConf.py - part 1
Fix all missing `etc/*.conf` errors permanently by copying `/etc/circos` into the run directory every time GenoVi generates `circos.conf`.

```
CONF="$HOME/.local/lib/python3.13/site-packages/scripts/createConf.py"
cp -a "$CONF" "$CONF.bak"

python3 - <<'PY'
import pathlib, re
p = pathlib.Path.home()/".local/lib/python3.13/site-packages/scripts/createConf.py"
txt = p.read_text()

anchor = 'file = open("circos.conf", "w")'
assert anchor in txt

txt = re.sub(
    r'\n\s*# Ensure Circos can resolve includes like etc/.*?\n',
    '\n',
    txt,
    flags=re.S
)

txt = txt.replace(anchor, anchor + '''
    import shutil, os
    shutil.rmtree("etc", ignore_errors=True)
    try:
        shutil.copytree("/etc/circos", "etc")
    except Exception:
        os.makedirs("etc", exist_ok=True)
''', 1)

p.write_text(txt)
PY
```

##### Patching of createConf.py - part 2
Prevent Circos “multivalue” errors caused by duplicate `svg`, `png`, and `file` directives.

```
python3 - <<'PY'
import pathlib, re
p = pathlib.Path.home()/".local/lib/python3.13/site-packages/scripts/createConf.py"
txt = p.read_text()

txt = re.sub(
    r'\n\s*# Force GenoVi/Circos to generate an SVG.*?pass\n',
    '\n',
    txt,
    flags=re.S
)

p.write_text(txt)
PY
```
